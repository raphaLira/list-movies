# Aplicação para listar filmes


## Setup

* Instale o NODEJS ( Versão LTS )

[NodeJS](https://nodejs.org/en/)

* Instale o YARN

```
$ npm i yarn -g
```


## Instalar as dependências

```
yarn install
```

## Para executar 

yarn start

Acessar em
[http://localhost:3000](http://localhost:3000) tvia browser.
